# Eshow

## 应用场景
eshow是一个亚马逊线上测评平台,亚马逊商家可以在该平台上发布任务,测评手可以在该平台上领取任务帮助商家完成测评,以获取相应的报酬.

## 相关技术

* 后台语言: Java 8
* 前端: Vue
* 数据库: Postgresql
* 后台框架: Springboot
